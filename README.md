# Todo App (Spring Boot)

Simple Todo application built with Spring Boot, Thymeleaf, Spring Data JPA and H2 in-memory database.

Run:

Windows / macOS / Linux (with Maven installed):

```bash
mvn spring-boot:run
```

Or build jar and run:

```bash
mvn package
java -jar target/todo-app-0.0.1-SNAPSHOT.jar
```

Open http://localhost:8080 in your browser.
